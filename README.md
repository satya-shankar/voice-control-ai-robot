# Voice Control AI Robot

## Introduction
This project is a voice-controlled robot using Arduino and speech recognition.

## Objectives
- Control robot using voice
- Wireless communication
- Easy human interaction

## Components Used
- Arduino UNO
- Bluetooth Module (HC-05)
- Motor Driver (L298N)
- DC Motors

## Working Principle
The robot receives voice commands via Bluetooth and processes them to control motor movement.

## Features
- Voice control
- Real-time response
- Wireless operation

## Applications
- Home automation
- Assistive robotics
- Smart control systems

## Conclusion
This project demonstrates how voice recognition can be used to control robotic systems effectively.
